Aktivitas 3 - Penyusunan Data

Rakitan data mentahmu menjadi tabel yang siap tempur

Di bengkel ini, kamu mengubah data hasil buruan menjadi tabel distribusi frekuensi. Setiap hitungan yang tepat akan mengaktifkan mesin pembentuk histogram.

Target aktivitas ini

  • Memahami minimum, maksimum, jangkauan, kelas, interval, dan frekuensi.
  • Menggunakan data sendiri untuk membangun tabel distribusi frekuensi.
  • Mengisi mesin tabel dengan interval dan frekuensi yang benar untuk membuka tantangan utama histogram.

Section 2

Rangkuman langkah membuat histogram

Gunakan rangkuman berikut sebagai penguatan setelah menonton video. Baca dari atas ke bawah karena langkah membuat histogram harus dilakukan secara urut.

Catatan penting: langkah membuat histogram harus dilakukan berurutan. Jika satu langkah terlewat, hasil tabel distribusi frekuensi dan histogram bisa menjadi tidak tepat.

Mengurutkan Data

Urutkan data dari nilai terkecil ke nilai terbesar agar sebaran data lebih mudah dibaca dan proses berikutnya lebih rapi.

Contoh: 150, 152, 154, 156, 160.

Menentukan Nilai Minimum dan Maksimum

Tentukan data terkecil sebagai minimum dan data terbesar sebagai maksimum. Kedua nilai ini menjadi dasar saat menghitung jangkauan.

Minimum menunjukkan batas awal data, sedangkan maksimum menunjukkan batas akhir data.

Menghitung Jangkauan Data

Hitung jangkauan untuk mengetahui lebar sebaran data dengan mengurangkan nilai maksimum dan minimum.

R = maksimum - minimum

Menentukan Banyak Kelas

Tentukan berapa banyak interval kelas yang akan dipakai. Pada materi ini, banyak kelas dapat dicari dengan Aturan Sturges.

k = 1 + 3,3 log n

Menentukan Panjang Interval Kelas

Hitung lebar setiap interval kelas dengan membagi jangkauan data oleh banyak kelas, lalu sesuaikan pembulatannya agar semua data dapat masuk.

i = R / k

Menyusun Interval Kelas

Susun interval kelas mulai dari nilai minimum, lalu lanjutkan secara berurutan sampai seluruh data tercakup tanpa ada kelas yang tumpang tindih.

Jika kelas pertama berakhir di 58, maka kelas berikutnya dimulai dari 59.

Menghitung Frekuensi Tiap Kelas

Masukkan setiap data ke interval yang sesuai, lalu hitung banyak data pada tiap interval untuk memperoleh frekuensi.

Jumlah seluruh frekuensi harus sama dengan jumlah data awal.

Amati data yang sudah diurutkan, hitung unsur pentingnya, lalu bentuk interval kelas agar tabel distribusi frekuensi dapat dibuat.
