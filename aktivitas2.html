Aktivitas 2 - Pengumpulan Data

Kumpulkan energi data dari dunia nyata

Di aktivitas ini, timmu berburu data numerik kontinu sebagai sumber energi utama untuk membangun histogram. Semakin rapi dan lengkap datanya, semakin mudah aktivitas berikutnya dibuka.

Target aktivitas ini

  • Pilih data yang bersifat numerik kontinu.
  • Gunakan data dari konteks nyata di lingkungan sekitar.
  • Kumpulkan minimal 25 data agar cukup untuk dianalisis.
  • Data yang dikumpulkan akan menjadi bahan bakar Aktivitas 3 dan Aktivitas 4.
Petunjuk Misi

Objective berburu data

SET
Tentukan variabel buruan

Pilih konteks data, nama variabel, dan satuan yang jelas sebelum mengisi angka.

25
Isi energy meter

Kumpulkan minimal 25 data valid agar meter aktivitas ini penuh.

REF
Laporkan alasan pilihan data

Tuliskan refleksi singkat tentang mengapa data ini cocok dibuat histogram.

Status Misi

Progress Aktivitas 2

Konteks Data

Pilih contoh konteks data atau tentukan konteksmu sendiri

Pilihan ini membantu kelompokmu membayangkan data yang cocok untuk disajikan dalam histogram.

Tinggi badan

Contoh data pengukuran tinggi badan murid dalam satuan cm.

Lama perjalanan

Contoh data waktu tempuh dari rumah ke sekolah dalam satuan menit.

Panjang langkah

Contoh data hasil pengukuran panjang langkah murid.

Denyut nadi

Contoh data denyut nadi setelah aktivitas ringan dalam satuan kali per menit.

Waktu menyelesaikan tugas

Contoh data waktu yang dibutuhkan murid menyelesaikan tugas sederhana.

Identitas Data

Tentukan variabel dan satuan data

Isilah identitas data sebelum mulai menambahkan angka hasil pengukuran atau observasi.

Gunakan nama variabel yang jelas dan mudah dipahami.

Satuan membantu data dibaca dengan tepat pada tahap berikutnya.

Pengumpulan Data

Tambahkan data satu per satu secara bertahap

Masukkan data numerik yang sudah kalian kumpulkan. Setiap data yang ditambahkan akan tampil sebagai kartu angka agar lebih mudah dibaca.

0 dari 25 data terkumpul

Mulailah menambahkan data hasil pengukuran atau observasi kelompokmu.

Data yang sudah terkumpul

Setiap angka yang ditambahkan akan muncul di area ini.

Ringkasan Data

Lihat data sebelum lanjut ke aktivitas berikutnya

Ringkasan ini membantu kelompokmu memahami data yang sudah dikumpulkan.

Refleksi Singkat

Pikirkan hubungan data dan histogram

Setelah melihat data yang terkumpul, jelaskan mengapa data tersebut lebih sesuai disajikan dalam bentuk histogram dibandingkan diagram batang.

Saat semua target selesai, aktivitas ini siap ditutup dan data akan dibawa ke Penyusunan Data pada Aktivitas 3.
